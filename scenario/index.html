剧情编辑教程


用剧情编辑器创建任务



剧情编辑器（ScenarioEditor）是提供给玩家编辑任务、剧情地图的工具。玩家可以通过该工具编辑适合多人对战的自定义剧情的地图，单人任务，甚至为游戏新增新的战役。

随后双击征服美洲游戏目录下的 ScenarioEditor.exe 可执行程序即可打开剧情编辑器，操作界面如下图所示。左边是添加、修改剧情语句的操作按钮，右边是已添加的剧情语句。

在对地图进行编辑之前，你需要通过游戏中自带的地图编辑器编辑一张地图并设置好需要用到的 任务组 和 任务区域，请参考本站 地图编辑器 板块。在剧情编辑器的 File 菜单中点击 New ，之后在打开的文件框中选择之前保存的 *.m3d 格式的地图文件。随后对其进行编辑添加剧情语句。编辑完后，在 File 菜单中点击 Save 即可保存游戏任务。该任务即会被添加到 单人游戏——单人任务 板块中。在 File 菜单中点击 Open 即可对已有的剧情文件进行编辑。编辑好的内容保存后，可以在 UserMissions 文件夹中查看。

注：因为 ScenarioEditor 功能不是非常完善，其中可以使用的单位判断仅包含了 征服美洲 v1.46 游戏中12个国家、文明的基本单位，征服美洲-反击 v1.50 中添加的5个国家的单位未被包含。并且，ScenarioEditor中也有不少BUG。关于如何解决这个缺陷，可以使用下面介绍的 用 Visual C++ 6.0 创建任务 的方法来创建剧情任务。

用 Visual C++ 6.0 创建任务

可以使用 Visual C++ 6.0 工具编辑任务SDK模板源代码来制作任务或战役。点击 MOD 工具下载 下载其中的 MISSIONS_SDK_AC.rar 文件。

对于编辑任务SDK源代码，可以不用完全掌握C++编程技术，任务SDK源代码中的一些语法相对简单，基本为一些脚本；当然，如果有C++编程基础当然更好。由于 征服美洲 系列游戏开发时间较早，2002～2003年期间开发，其所有的C++源代码 只能在 Visual C++ 6.0 版本上进行编译。推荐安装目前使用较广的 Visual C++ 6.0（完整绿色版）。
