《征服美洲》系列的开发历史

《征服美洲》(American Conquest)系列游戏是由乌克兰的GSC Game World开发、德国的CDV Software Entertainment 游戏公司发行的运行于微软Windows操作平台上的历史类即时战略游戏。第一款《征服美洲》版本采用的是DMCR v1.46 游戏引擎 (DoomCraft引擎) ，发布于2002年11月18日，游戏背景被设定在1492～1813年的美洲大陆。游戏中有12个不同的国家和文明，其中有西班牙(Spain)、英国(Britain)这种典型的西欧殖民国家，也有玛雅(Maya)、印加(Incas)等美洲古代文明，当然也不会缺少在北美大陆谱写了可歌可泣的历史故事的部落民族。游戏的官方版本是英文版，同时发行商也与若干国家的游戏代理商发行了德文、法文、俄文、意大利文、波兰文、日文等版本。交大铭泰（北京）软件有限公司旗下的东方娱动科技有限公司在跳票近半年后，于2003年12月24日在中国大陆地区发布了《征服美洲》的简体中文版。
2003年6月，GSC Game World推出了《征服美洲》系列的第一个资料片扩展，即《征服美洲：反击》(American Conquest : Fight Back)，该版本采用了DMCR v1.50 引擎。在游戏内容上，《征服美洲：反击》增加了俄罗斯(Russia)、荷兰(Netherlands)、海达(Haida)5个新国家（文明）与50多个新单位。
2006年1月，第二个资料片《征服美洲：分裂的国家》(American Conquest: Divided Nation)由CDV Software Entertainment 正式发布。这款资料片版本采用了 DMCR v1.86 游戏引擎，它几乎完全抛弃了上两个版本的游戏内容，把历史背景锁定在了从到1812～1863年间的北美地区；游戏中有4个主要国家：美利坚联邦(The Union)、美利坚邦联(The Confederacy)、德克萨斯共和国(Republic of Texas)和墨西哥(Mexico)，以及120多个全新单位。光谱咨询股份有限公司与 CDV Software Entertainment 合作在台湾代理发行了繁体中文版，该版本将游戏标题译为《征服美洲：分裂的国土》。
《征服美洲：黄金版》(American Conquest - Gold Edition) 是由“征服美洲大中华区游戏平台”网站多名游戏玩家制作的整合版本，该版本采用了DMCR v1.86 游戏引擎。在游戏内容上，该版本将《征服美洲》和《征服美洲：反击》及玩家制作的《东亚崛起MOD》整合，增加了大明(Ming)、日本(Japan)、朝鲜(Korea)和纳奇兹(Natchez) 4个国家（文明），新增了近60多种新兵种、单位、船只，以及40多个东亚风格、北美原住民风格的建筑；除了新增了一些亚洲特色的单位外，还有一些例如神机箭等特色的大型单位。《征服美洲：黄金版》虚构了大明、朝鲜、日本等东亚国家殖民美洲的历史，同时也增加了若干虚构的战役。《征服美洲：黄金版》于2020年3月通过本网站发布。


GSC Game World 简介

GSC Game World成立于1995年，由谢尔盖·格里戈洛维奇·康斯坦丁洛维奇(英语:Sergey Grigorovich Constantinovich，乌克兰语:Сергій Костянтинович Григорович) 创立于乌克兰基辅。1990年，谢尔盖尚在读书时就想好了公司的名称和Logo，GSC这一名称包含了其名称的缩写。GSC Game World是乌克兰最著名的游戏开发工作室，也是欧洲知名游戏开发公司。其下还有一家GSC World Publishing游戏发行公司。
早期发展
谢尔盖·格里戈洛维奇组建了一个15人的游戏工作室，并将一间两室的公寓作为办公地点，早期的员工包括谢尔盖的弟弟叶夫根尼·格里戈洛维奇(Evgeniy Grygorovych) ；当时谢尔盖只有17岁。在之后的发展中，公司不断扩展，顶峰时人数达到了200人。
最初，GSC Game World的工作将其他游戏进行语言本地化。1997年，公司开始研发冒险类游戏《追求》(Quest)，但由于缺乏经验，该项目被很快放弃。
在1998年俄罗斯发生经济危机之后，东欧游戏市场开始萎缩，GSC Game World 转向当时较为热门的即时战略类游戏的开发。同年，公司完成了第一款非营利性的游戏《魔兽争霸2000：核瘟疫》(WarCraft 2000: Nuclear Epidemic)。该游戏翻版了《魔兽争霸II》(Warcraft II)，甚至在一些方面超越了原作，GSC籍此寻求与暴雪娱乐 (Blizzard Entertainment, Inc.)的合作，希望合作开发《魔兽争霸III》，但暴雪对此并没什么兴趣。《魔兽争霸2000：核瘟疫》在游戏市场虽未取得成功，但为此研发的 DoomCraft (DMCR) 图像引擎在后续的游戏开发中发挥了很大作用，《哥萨克》、《征服美洲》、《亚历山大大帝》、《哥萨克2》、《灭国英雄》游戏都是用到了这款引擎。GSC甚至还将该引擎卖给其他游戏公司。
辉煌时期
1999年，GSC Game World 开始了著名的《哥萨克》系列游戏的开发。2001年4月24日，《哥萨克：欧洲战争》(Cossacks: European Wars)正式发布，该游戏在市场上获得巨大成功。《哥萨克》系列的开发主要由乌克兰本土开发人员完成，当时谢尔盖想开发一些爱国题材的游戏，就像一提到武士就想到日本一样，“哥萨克”这个称呼很容易让人联想到乌克兰。同年又发行了一款第一人称射击游戏《毒液. 代号：爆发》(Venom. Codename: Outbreak)。2002年，《哥萨克》系列的资料片《哥萨克：战争艺术》（Cossacks: The Art of War）发布。同年底，GSC Game World发布了《哥萨克》系列的资料片《哥萨克：重返战争》(Cossacks: Back to War)和《征服美洲》。《哥萨克》系列游戏全球发售量超过500万份。同时，公司也进入了《潜行者》(S.T.A.L.K.E.R.)系列游戏的研发。2002~2003年，公司又发布了《征服美洲：反击》，以及街机竞速游戏《盘旋高手》(Hover Ace)与射击游戏《纵火者》(FireStarter)三款游戏。除了《征服美洲》系列同样获得了游戏媒体的大量好评外，另外两款游戏很快被人遗忘了。
由于政策上的原因，GSC Game World之前的游戏都由其他机构发行，例如《哥萨克》与《征服美洲》系列都由德国CDV软件娱乐公司(CDV Software Entertainment AG)公司发行。2004年，GSC Game World成立了自己的出版机构 GSC World Publishing，并与育碧(Ubisoft)合作发行了一款采用DMCR引擎3D升级版的即时战略游戏：《亚历山大大帝》(Alexander)，该游戏由2004年科林·法瑞尔(Colin Farrell)主演、奥利弗·斯通(Oliver Stone)导演的同名电影正式授权的。进入2005年，GSC Game World带来了与《亚历山大大帝》同样游戏引擎的《哥萨克》系列游戏续作——《哥萨克2：拿破仑战争》(Cossacks II: Napoleonic Wars)。2006年又开发了《哥萨克2》的资料片，《哥萨克2：欧洲战争》(Cossacks II: Battle for Europe)与奇幻背景的即时战略游戏《灭国英雄》(Heroes of Annihilated Empires)。尽管游戏媒体对《哥萨克2》系列游戏普遍表示赞赏，但该作在玩家群体中依然存在争议，发行量不如前作《哥萨克》系列；《灭国英雄》可以说是GSC对早期计划的“魔兽争霸III”项目的实现，开发人员在游戏中大胆的加入了RPG元素，采用了奇幻背景，但由于自身的一些缺点、且当时的游戏市场竞争激烈，该作同样未取得预期成功。
在2006年，由德国CDV软件娱乐公司主导，GSC与德国Revolution of Strategy工作室合作开发了《征服美洲：分裂的国家》。由于开发时间较短及一些经济纠纷，《征服美洲：分裂的国家》在市场及媒体评价上并未取得较好的成绩。该作由CDV软件娱乐公司全权发行，因此CDV倒闭后，GSC也无权发行《征服美洲：分裂的国家》的数字版本。在2007年，CDV软件娱乐公司还将《征服美洲》系列三个版本打包在一起推出了一个合集版本，但合集版本的内容依然是《征服美洲》和两个资料片三个独立的游戏。
在多年的运作中，GSC Game World与全球顶尖发行商及俄罗斯发行商多有合作，包括美国THQ游戏公司(THQ Inc.)、法国育碧娱乐公司、奧地利科氏傳媒 (Koch Media)、德国CDV软件娱乐公司、俄罗斯 1C等。
2007年3月20日，废土射击题材的《潜行者：切尔诺贝利的阴影》(S.T.A.L.K.E.R: Shadow of Chernobyl)发布并受到一致好评，销量也名列前茅，该游戏成为了当时GSC Game World最成功的项目；游戏背景为切尔诺贝利核事故后的废弃地区。GSC World Publishing负责在独联体国家发行游戏，而THQ游戏公司负责全球发行。同是这一年，公司也开发了基于Java ME手机平台的《潜行者》游戏。2008年，该公司又推出资料片《潜行者：晴空》(S.T.A.L.K.E.R: Clear Sky)。2009年，《潜行者》系列第二个资料片《潜行者：普里皮亚季的召唤》(S.T.A.L.K.E.R.: Call of Pripyat) 发布，同时谢尔盖·格里戈洛维奇宣布将继续开发《潜行者》系列游戏的第二部，并计于2012年发布；在当时《潜行者》系列三部曲成为了废土射击系列游戏的经典之作。《潜行者》项目早于2001年就已经开始，当时公司以Open Dynamics Engine游戏引擎为基础开发了X-Ray3D引擎；最初，该游戏被命名为《遗忘迷失》(Oblivion Lost)，游戏背景是一群科研人员使用互联网进行世界旅行，但外人并不看好这个项目。之后，开发人员受到小说《路边野餐》(英语:Roadside Picnic ，俄语:Пикник на обочине)及根据小说改编的科幻电影《潜行者》(Сталкер)的启发，决定将游戏背景设为核灾难后的切尔诺贝利并改了名字叫做《潜行者：遗忘迷失》；开发人员甚至还获得政府许可进入了切尔诺贝利禁区取材；之后该项目就一直受到各界关注，多家发行商主动与GSC联系争取发行。该游戏原定于2004年秋季发布，早在2003年，《潜行者》的早期版本由于开发人员的粗心大意而遭到泄漏，最终它在2007年才问世，并采用了现在的游戏名字。《潜行者》游戏在发售两周内就售出100万份，算上随后推出的资料片，系列全球发行量超1000万份，该作让GSC在世界顶尖游戏商中占据了一席，且在一些大型国际游戏展览会上多次获奖，并获得了媒体及玩家的高度评价；游戏获得当年“年度最佳游戏”及“最佳射击游戏”称号，游戏销售数量也相当出色。在2009年时，GSC Game World是东欧最大的游戏公司，金融服务公司安永会计师事务所（Ernst＆Young）在2011年2月将谢尔盖评为“年度企业家”。仅在独联体国家，遵循进一步发展品牌的战略，GSC Game World于2007年发行了一系列基于《潜行者》的小说（俄语和德语版），总销量超过500万本。
停滞阶段
2011年12月9日，由于陷入了发展困境。谢尔盖·格里戈洛维奇宣布GSC Game World解散，宣称这是由于个人原因，但具体原因不明，有传闻公司可能遇到财务危机，甚至有传闻说是克兰当局也有参与此事。之后，GSC Game World作为一家公司继续存在，但并没有任何经营，《潜行者2》系列与《灭国英雄2》的开发也终止，公司将给员工工资支付到2012年2月后正式停止运营；据说部分员工对管理层产生不满，并在《潜行者2》游戏的开发上产生的分歧。不久后，《潜行者2》系列的开发资料在互联网上不同程度的泄漏。随后，谢尔盖·格里戈洛维奇在接受采访时表示，其管理层正在解决问题以维持公司存续，但会辞退掉与管理层分歧较大的员工；另外谢尔盖还表示公司依然在解决《潜行者2》的开发困难。
重生
2014年12月22日，GSC Game World重新开始工作，由谢尔盖·格兰格里洛维奇与其兄弟叶夫根尼共同管理。2015年5月18日，公司宣布开发《哥萨克》系列游戏的最新作品——《哥萨克3》(Cossacks 3)。此时，游戏市场中的即时战略游戏都在走下坡路，很少有游戏开发商去开发新的即时战略游戏。《哥萨克3》正式发布于2016年9月21日，游戏采用了全新的 3D引擎，这是该游戏一大亮点；除了新增大量优秀的历史战役，游戏内容大多是对初代《哥萨克》游戏的重置，且在游戏体验比较一般，甚至一些初代《哥萨克》中就存在的问题在《哥萨克3》中依然存在。游戏媒体多对其打出了及格成绩，评价其为“依赖怀旧情怀且毫无创新之作”。GSC表示他们会与玩家社区紧密合作，并致力于添加更多的游戏内容。之后，GSC为《哥萨克3》开发了4个DLC和1个扩展包，新增了苏格兰、葡萄牙、丹麦、荷兰、瑞士等多个可选择的国家及新的战役。
2018年5月，谢尔盖在其Facebook账户上发布文章，宣布重启《潜行者2》项目的开发。
注：本文记载内容截止2020年。


CDV Software Entertainment 简介

CDV软件娱乐公司（CDV Software Entertainment AG）是一家德国电子游戏发行商，成立于1989年，由Wolfgang Gäbler和Christina Oppermann在卡尔斯鲁厄（Karlsruhe）创立，隶属CDV软件娱乐集团（CDV Software Entertainment Group）。
2000年4月7日，CDV于法兰克福股票交易所上市。在2000年代初期，CDV是德国电子游戏市场上最大的出版商。 作为发行商，CDV通常与第三方开发公司或工作室合作，例如GSC Game World和Revolution of Strategy，CDV获取该开发商某款游戏的全球发行权，并由CDV指导开发，并负责市场营销和销售；同时CDV也负责项目融资和管理，对此进行的营销活动包括：产品设计和包装、在线与媒体的广告、向消费者直接推广、网站设计、与其他国家游戏公司共同开发本地化语言的游戏版本等。
自公司成立以来，发行了不少当时流行的游戏。例如《突袭》(Hidden Stroke)系列游戏。2000年，公司正式上市；2001年，CDV成为德国最大的PC游戏发行商，市场份额占4.6%。从2002年开始，由于经营失利，公司开始出现亏损；2003年，一些游戏的销售未达到预期，资金短缺，大量裁员。好在2004年开始，财务状况重新回暖。但随着2005年期间《哥萨克2》系列游戏销量不佳，财务状况再次吃紧，《征服美洲：分裂的国家》在此时也启动开发。
这期间，两位创始人Wolfgang Gäbler和Christina Oppermann的股份也不断减持，从2003年的54.14%减少到2007年的9.11%。 公司股票跌至最低点，同年财务报表显示亏损超过820万欧元。之后，由于一系列为稳定股价而进行的措施的失败后果，最后公司中仅有少量员工还保持积极性，并且股票价格始终未能再次涨上去。2010年4月12日，CDV 软件娱乐公司申请破产。


Revolution of Strategy 简介

Revolution of Strategy原是以Peter Hoffmann为首的位于德国的《突袭》游戏MOD制作组，最早他们制作了《突袭》APRM MOD（Allied Power Realism Mod，写实盟军军力MOD）。
2003年，CDV软件娱乐公司与之合作开发《突袭》系列游戏扩展包，玩家成为了开发者，MOD制作组也转为公司运营。Revolution of Strategy于2003年和2005年分别开发了《秘密行动：APRM》(Hidden Stroke: APRM) 和《秘密行动II：APRM》(Hidden Stroke II: APRM)。
2005年上半年，Revolution of Strategy与CDV合作开发《征服美洲：分裂的国家》，由Peter Hoffmann和Shaun Fletcher牵头。经过半年多的紧张开发，《征服美洲：分裂的国家》于2006初发布。
此时CDV软件娱乐公司进入发展瓶颈，财务状况较差。Revolution of Strategy与CDV也陷入经济纠纷，最后上升至法律层面。但随着CDV的破产，Revolution of Strategy最终没有取得任何开发费用。
